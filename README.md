## ðŸ§  Methodology

The project employed **LLM-mediated comparative reasoning**:
1. Load two campaign concepts (A/B).  
2. Simulate structured evaluation through `query_LLM()` calls.  
3. Compare narrative, tone, visual identity, and emotional effect.  
4. Merge strongest attributes into a hybrid product.  
5. Output final recommendations and campaign materials.

This process demonstrates how **AI dialogue** can serve as a co-creative partner, not just an evaluator.
